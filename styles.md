# KASHLET — STYLES GUIDE

## 1. Visual Direction

Kashlet is a calm, structured financial workspace designed for clarity, speed, and trust.

The UI should feel:
- professional but not corporate
- minimal but not empty
- structured but not rigid
- modern but not flashy

Inspired by:
- Linear (layout + spacing discipline)
- Notion (data organization)
- Stripe Dashboard (financial clarity)
- Apple Wallet (trust + simplicity)

---

## 2. Color System (NO GRADIENTS)

### Base Colors
- Background: #0B0F14 (deep dark neutral)
- Surface: #111827
- Card: #161B22

### Text Colors
- Primary text: #F9FAFB
- Secondary text: #9CA3AF
- Muted text: #6B7280

### Accent Color (ONLY ONE PRIMARY)
- Primary: #4F46E5 (indigo, restrained usage)

### Status Colors
- Success: #22C55E
- Warning: #F59E0B
- Error: #EF4444
- Info: #38BDF8

NO gradients anywhere in the system.

---

## 3. Typography System

### Font Family
- Headings: Space Grotesk
- Body: Plus Jakarta Sans

### Type Scale

- Display: 32px / 40px
- H1: 28px / 36px
- H2: 24px / 32px
- H3: 20px / 28px
- Body L: 16px / 24px
- Body M: 14px / 20px
- Caption: 12px / 16px
- Label: 12px / 16px (medium weight)

---

## 4. Layout System

- Grid: 12-column desktop layout
- Max width: 1440px
- Content width: 1200px
- Spacing scale:
  4px / 8px / 12px / 16px / 24px / 32px / 48px / 64px

---

## 5. Border Radius

- Small: 6px
- Medium: 10px
- Large: 14px
- Cards: 12px

No overly rounded “bubble” UI.

---

## 6. Buttons

### Primary Button
- Height: 40px
- Padding: 12px 16px
- Font: 14px medium
- Background: Primary color
- Text: white

### Secondary Button
- Border: 1px solid #2A2F3A
- Transparent background

### Ghost Button
- No background
- Hover: subtle surface highlight

---

## 7. Input Fields

- Height: 40px
- Background: #0F141B
- Border: 1px solid #2A2F3A
- Focus state: primary color border
- No shadows

---

## 8. Tables (Core to Product)

- Stripe-like density
- Alternating subtle row hover
- Clear column separation
- Compact spacing for financial data

---

## 9. Cards

- Flat design only
- Subtle border
- No shadows
- Hover: slight surface lift only

---

## 10. Icon Style

- Lucide / Feather style
- Outline icons only
- No filled icons except status indicators

---

## 11. Interaction Rules

- Hover states: slightly lighter surface
- Active states: accent border highlight
- No animations longer than 200ms
- No flashy transitions

---

## 12. Data Visualization

- Simple bar indicators
- No complex charts
- Focus on readability over aesthetics

---

## 13. UX Tone

Kashlet should feel:
- calm
- precise
- structured
- fast

Never:
- playful
- noisy
- decorative
